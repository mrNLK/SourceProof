# 18 — Add Candidate Notes/Tags

Add the ability to attach free-text notes and tags to any candidate. On the candidate card expanded view, add a notes textarea (auto-saves on blur) and a tag input (type + enter to add, click X to remove). Predefined tag suggestions: "Strong Fit", "Backup", "Refer to Other Role", "Culture Add", "Overqualified", "Underleveled". Notes and tags should be visible on pipeline cards as small badges. Persist in the same store as pipeline data.
