# 14 — Add Outreach Copy-to-Clipboard with Tracking

When the user clicks "Generate Outreach" on a candidate, after the message generates: (1) show a "Copy" button next to the message, (2) clicking Copy saves to clipboard and shows a brief "Copied" toast, (3) automatically move the candidate to "Contacted" stage in Pipeline if they're already in the pipeline, (4) if not in pipeline, prompt "Add to pipeline as Contacted?" with a one-click confirm. This closes the loop between outreach generation and pipeline tracking.
