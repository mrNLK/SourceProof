# 13 — Add Keyboard Shortcuts

Add keyboard shortcuts for power users: (1) Cmd+K or Ctrl+K opens a command palette with search for tabs, actions, and recent searches, (2) 1-6 number keys switch between tabs when command palette is open, (3) Escape closes any modal or expanded card, (4) Arrow keys navigate between candidate cards in Results, (5) S stars/unstars the focused candidate, (6) P adds the focused candidate to pipeline. Show a small "?" icon in the bottom-right that displays the shortcut reference when clicked.
