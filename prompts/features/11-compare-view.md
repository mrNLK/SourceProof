# 11 — Add Candidate Comparison View

Add a "Compare" mode to the Results tab. Users can checkbox 2-3 candidates, then click "Compare" to open a side-by-side table view. Columns: candidate name (header), score, top 3 repos, languages, stars, hidden gem status, EEA highlights. Table should be horizontally scrollable if needed. Include a "Send to Slack" button that formats the comparison as a clean markdown table and copies to clipboard. This is separate from the Bulk Actions compare -- this is a quick inline compare on the Results tab.
