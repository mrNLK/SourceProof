# 06 — Validate Inputs Before Strategy Generation

Add input validation to the New Search form. Rules: (1) Role field is required and must be at least 3 characters, (2) Company field is required for Path 1, (3) JD text must be at least 50 characters for Path 2, (4) URL must match a valid URL pattern for Path 3. Show inline validation errors below each field in red. Do not allow form submission until validation passes. Do not use browser-native validation -- use custom inline error messages styled consistently with the app.
